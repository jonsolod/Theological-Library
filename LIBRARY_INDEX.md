# Theological Library — External Index
## For Claude: How to Access the Full Library

This index lists all documents in the GitHub theological library.
When a user asks about a topic covered by a GitHub document,
fetch it using web_fetch on the raw GitHub URL below.

**Repository:** https://github.com/jonsolod/Theological-Library
**GitHub Pages:** https://jonsolod.github.io/Theological-Library
**Raw Base URL:** https://raw.githubusercontent.com/jonsolod/Theological-Library/main

---

## HOW TO USE THIS INDEX

1. Search project knowledge first — NASB, CCC, councils, Fathers are already here
2. If a needed document is NOT in project knowledge, check this index
3. Fetch it with web_fetch using the Full URL column below
4. The full text will be available for the conversation
